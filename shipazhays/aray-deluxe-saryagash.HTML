Описание санатория

Санаторий Арай Делюкс находится внутри парковой зоны курорта Сарыагаш профсоюзный. Элитный санаторий позиционируется как гостиница-санаторий уровня 5 звезд. Есть парковка, рядом магазины, кафе, ресторан тоже недалеко. Санаторий оснащен двумя бассейнами, сауной, лечебным корпусом.

Номера

Номера в коттедже Эконом класса. В основном корпусе номера Премиум, Премиум Люкс, Премиум Делюкс и Роял Сьют.

В номере: халат х/б, чистый. Полотенце для лица и банное полотенце.

Питание

Питание для гостей раздельное. Для особых гостей и общий зал.

Паркинг для автомобилей имеется на территории парка. Мест для парковки предостаточно.

Рядом с санаторием есть кафе Рахат. Несколько продуктовых магазинов рядом с санаторием (за территорией).

Для базарных покупок можете посетить Сарыагаш. Там одежда по оптовым ценам. Продукты. В парке сухофрукты продаются по приемлемым ценам.

Трансфер из Шымкента производится с аэропорта, из ЖД вокзала или же с другой точки города.

Wi-Fi имеется, скорость хорошая.

Доставка суши, пицца, чикен, лаваш, шаурмы.

Аптека находится возле поворота на курорт, или же в центре курорта. Цены не особо высокие. Средние цены, ассортимент довольно широкий.
